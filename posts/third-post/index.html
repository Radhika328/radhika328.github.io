The nature of information problem

February 18, 2026

In this unit, I learned that images can also be seen with a perspective of data and math. I learned how to transform and optimize images by seeing them as changeable information. For me photos were stuff which I can store them as memory, but this unit changed my perspective. I started taking images in a more technical way, and I started describing them using math and data. 

This unit made me realize that the image is made up of thousands of tiny units called pixels. Each pixel stores information about brightness and color. By combining these pixels, we get images which we see on our phones, laptops, etc. I also learned about the RGB color model. Which stands for Red, Green and Blue.It means these colors mix in different amounts to create many other colors in a digital image. I related it with the color wheel concept in painting which says Red, Blue and Yellow are primary colors. By mixing them in different amounts we get all other colors.By using this idea, I can change the RGB values to modify my image.

On the other hand, I also understood that I can change the appearance of images by changing pixel values. I can modify an image by changing color, brightness, and by even transforming a particular part of an image. By learning about these concepts, I understood how editing apps works. In the beginning, optimizing concepts was a bit confusing to me as I never studied it before and because it emphasizes having the best possible result with less resources. For example, it tends to improve image quality while maintaining file size. But by practicing adding pixel values and by adding simple functions, I was able to understand this concept better.
