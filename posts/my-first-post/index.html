The nature of information problem

February 18, 2026

In this unit, I learned about transforming and optimizing images. At first, the topic felt a bit challenging because I had never considered images this way before. Previously, I only viewed images as pictures on my phone, laptop, or social media. I didn’t realize that images can also be described using math and data. This topic helped me see that images are not just pictures; they represent information that can be changed and improved.

One important thing I learned is that an image consists of many small units called pixels. Each pixel holds data about color and brightness. When many pixels come together, they create the complete image we see on the screen. We also learned about the RGB color model, which stands for Red, Green, and Blue. These three colors mix in different amounts to produce many other colors in a digital image. Now, I realize that adjusting these RGB values can change the color and appearance of an image.

Image transformation involves changing the pixel values to improve or modify an image. For example, we can increase brightness, adjust contrast, or highlight certain parts of an image. This helped me understand how photo editing apps function when we apply filters or make adjustments. The optimization part was new to me. At first, it was a bit confusing because it focuses on achieving the best results while using limited resources, like maintaining image quality while reducing file size. Doing small exercises, such as adjusting pixel values or applying simple functions to images, helped me grasp the concept better.
